# 👶 胎动记录器

> 专业孕妇胎动记录应用

## 功能特点

- 📱 一键记录宝宝胎动
- 📊 实时统计与分析
- ☁️ 云端数据同步
- 🎨 专业简洁的UI设计

## 技术架构

### 新架构（v2.0）

```
├── index.html          # 主页面（760行）
├── js/
│   ├── config.js       # 配置文件
│   ├── storage.js     # 本地存储
│   ├── auth.js        # 用户认证
│   ├── data.js        # 数据库操作
│   └── app.js         # 主应用
└── css/
    └── styles.css     # 样式文件
```

### 旧架构（v1.x）

单文件架构，所有代码集成在 `index.html` 中。

## 开发

```bash
# 安装依赖
npm install

# 开发
# 编辑 index.html 和 js/*.js

# 部署
./deploy.sh
```
