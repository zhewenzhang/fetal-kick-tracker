# 胎动记录器 👶

记录宝宝胎动的可爱工具

## 访问方式

**Web版**（推荐）：https://zhewenzhang.github.io/fetal-kick-tracker/

## 功能

### 🎯 核心功能
- 👆 一键记录胎动
- 📊 今日/本周统计
- 📅 历史记录查询
- 💾 本地存储（隐私安全）

### 🤰 孕期感知
- 设置预产期，自动计算孕周
- 根据孕期显示正常范围
- 智能状态评估（正常/略少/偏少）
- 孕期建议提示

### 📈 计数模式
- **快速计数**：随时点击记录
- **定时计数**：2小时计时模式
- 医学标准参考

### 📋 报告
- 每日详细报告
- 12小时等效计算
- 高峰时段分析

## 使用指南

### 1. 打开网页
手机浏览器访问：https://zhewenzhang.github.io/fetal-kick-tracker/

### 2. 添加到主屏幕（推荐）
- **iPhone (Safari)**：分享按钮 → 添加到主屏幕
- **Android (Chrome)**：菜单 → 添加到主屏幕

### 3. 设置
1. 点击右下角「设置」
2. 输入预产期
3. 开启/关闭震动提示

### 4. 记录胎动
- 点击中间大按钮记录胎动
- 可切换「定时计数」模式

## 孕期标准参考

| 孕周 | 2小时标准 | 1小时标准 | 特点 |
|-----|----------|----------|------|
| 16-20周 | ≥6次 | ≥2次 | 开始感知 |
| 20-28周 | ≥6次 | ≥3次 | 逐渐规律 |
| **28-34周** | **≥10次** | **≥4次** | ⭐最活跃 |
| 34-37周 | ≥8次 | ≥3次 | 略减少 |
| 37-40周 | ≥6次 | ≥2次 | 准备分娩 |

## 数据说明

- **存储位置**：本地浏览器（localStorage）
- **数据安全**：不上传云端，隐私保护
- **导出支持**：支持导出JSON备份
